## ⚙️ 설치 환경 실행 방법

- 모듈 설치

해당 프로젝트는 `vite`를 사용하였으며 `vite`는 Node version 14.18, 16+ 이상일 경우에만 지원합니다.
프로젝트 실행시 가능하면 .nvmrc에 있는 node version을 사용해주세요.


```
yarn
```

- 실행

```
yarn dev
```

- 배포 환경

```
yarn build
yarn start
```
